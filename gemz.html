GEMZ

Sparkling loyalty and engagement - The dynamic reward staking APP

GEMZ ia a short project to design a reward page, with a loyalty program that allows users to earn points and redeem rewards, including fine dining experiences, live sports event tickets, and music concert passes. To enhance user engagement, the reward page incorporates dynamic animations of gems stacking up, with 21 blocks representing each loyalty level.

About this project

My role : Individual
User experience / User Interface/

Main tools involved
Figma

Project duration
1 week

Starting from the user

Persona : Understanding User Needs

Mark, is a 35-year-old tech-savvy IT Project Manager. He enjoys dining out at fine restaurants, attending sports events, and going to concerts to discover new experiences. Mark values convenience and seeks meaningful rewards to enhance his lifestyle. He's proficient with technology but tends to be skeptical of apps, preferring to use his smartphone for essential tasks.

Key features

After understanding the persona, I designed GEMZ with an eye catching, playful and progressive reward system.

Dynamic Gem Stacking: The app features a visually engaging dynamic gem stacking animation that displays the user's progress and rewards in a fun and interactive way.

Tailored Rewards: GEMZ offers a diverse range of rewards to align with the user's interests. The interface is designed for easy navigation, so that the user can quickly find, track, and redeem rewards.

Status Levels: Users can achieve different status levels within the loyalty program, offering a sense of achievement and recognition.

Prototype Design

Multiple frames and animated components are incorporated to deliver a dynamic and engaging user experience, enhancing the user's interaction with the app.

Outcome expectation

By tailoring the GEMZ app to address the user's specific goals, motivations, challenges, and personality traits, we anticipate several positive outcomes:

Increased User Engagement: dynamic gem stacking animation, along with animated components, will capture the user's attention and keep him engaged with the app.

Enhanced User Experience: app's streamlined interface, visual rewards tracking, and animations will make it more enjoyable for Mark to save money and discover new experiences.

Strong First Impression: addressing the user's app loyalty tendencies, GEMZ will make a strong first impression and encourage him to stick with the app. This updated section emphasizes the use of animated components to enhance the user experience within the GEMZ app.

Improved Loyalty: find it easier to engage with the loyalty program, fostering a sense of loyalty and recognition.
